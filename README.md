# 👥 Friend Tracker App

## 📌 Project Description

Friend Tracker is a simple and effective web application that helps you maintain meaningful relationships by keeping track of your friends and how often you stay in touch with them. It reminds you when it's time to reconnect, ensuring no important connection fades away.

---

## 🚀 Technologies Used

* Next.js
* React.js
* Tailwind CSS
* DaisyUI
* JSON (for local data storage)

---

## ✨ Key Features

### 📋 1. Friend List Dashboard

View all your friends in a clean and responsive card layout with essential information like contact status, tags, and last interaction time.

### 🔗 2. Dynamic Friend Details Page

Click on any friend card to navigate to a detailed profile page showing bio, contact goals, and next due date.

### 🎯 3. Smart Status Tracking

Each friend is categorized as:

* 🔴 Overdue
* 🟡 Almost Due
* 🟢 On Track
  based on your communication frequency goals.
